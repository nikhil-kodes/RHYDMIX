# RHYDMIX

## About RHYDMIX!
RHYDMIX is a frontend project which is made using HTML, CSS and JavaScript. It suggests music to users based on the weather in their location and several inputs from the users which include their mood, their choice of genre and also their relationship statusüòÅ. It also takes the language preference from the user for the type of song selection.

## How RHYDMIX works!
RHYDMIX uses Geolocation API to take the location co-ordinates of the user and Weather API to determine the weather of that particular area, it then takes few inputs from the users based on their preferences and by using Spotify API and it fetches song recommendations for the user.
